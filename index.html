Введення у мову паскаль

Я не буду зараз писати як і чому вона була створена, для цього є вікідепія, я лише буду казати про синтаксис данної мови

Кожна программа на цій мові починається з

program Example;
begin
end.

Де замість Example - назва вашої програми. Код пишеться між ключовими словами begin і end, але є виключення, наприклад оголошення змінної, але про це пізніше. Змінні - це дуже крута штука, без яких не напишеш серйозної программи. Змінна - це величина, яка може змінюватись по ходу програми. у них можна записувати як і слова, так і числові дані На мові паскаль вони оголошуються після ключового слова "var", ось так:

program Example;
var
zminna : String;

begin
end.

Зверніть увагу, що після назви нашої змінної (zminna) через дві крапки щось написано. Це - тип данних. Їх у мові паскаль є багато, але для початку ми будемо використовувати тільки декілька з них
String - це тип данних, у якому зберігаються стрічки та слова (наприклад 'Тут був вася')
Integer - це тип данних, у якому зберігаються числові типи данних (наприклад 377262)
Real - теж числовий тип данних, але у ньому зберігаються числа з крапкою (наприклад 3.1415)

А також є незвичайний тип данних - Boolean. У ньому не зберігаються числа чи слова, він приймає тільки два значення: true і false.

Присвоєння значень змінній

Про те, що можна присвоїти змінній ми вже знаємо, але як це зробити? Для цього є оператор присвоєння ":=" Розглянемо на прикладі:

program Example;
var
zminna : String
begin
zminna := 'Наша стрічка';
WriteLn(zminna);
ReadLn();
end.
